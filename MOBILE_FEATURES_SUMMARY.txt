OPTION 1: Browser DevTools (Recommended for quick testing)
──────────────────────────────────────────────────────────
1. Start app:     streamlit run app.py
2. Open browser:  http://localhost:8501
3. Press F12:     Open DevTools
4. Toggle device: Ctrl+Shift+M (Windows) / Cmd+Shift+M (Mac)
5. Select device: iPhone 12, iPad, Galaxy S20, etc.
6. Test features: Scroll, tap buttons, check layout

OPTION 2: Real Mobile Device (Best for final testing)
──────────────────────────────────────────────────────
1. Find your IP:  ifconfig | grep "inet " (Mac/Linux)
                  ipconfig (Windows)
2. Start app:     streamlit run app.py --server.address 0.0.0.0
3. On phone:      Open browser → http://YOUR_IP:8501
4. Test all:      Navigation, buttons, scrolling, forms

OPTION 3: Responsive Design Mode (Safari)
──────────────────────────────────────────
1. Safari:        Develop → Enter Responsive Design Mode
2. Select:        Device preset (iPhone, iPad, etc.)
3. Test:          All interactive features
